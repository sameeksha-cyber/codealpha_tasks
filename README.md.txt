# Network Intrusion Detection System (IDS)

## Project Description

This project is a basic Network Intrusion Detection System developed using Python and Scapy. It captures live network packets, detects TCP, UDP, and ICMP protocols, displays source and destination IP addresses, and stores packet information in a log file.

## Features

- Live Packet Sniffing
- TCP Detection
- UDP Detection
- ICMP Detection
- Source & Destination IP Monitoring
- Packet Logging

## Technologies Used

- Python
- Scapy
- VS Code
- Wireshark

## Installation

Install Scapy:

```bash
pip install scapy
```

Run the project:

```bash
python ids.py
```
